nPoco a poco, y fruto de varias conferencias (ver WEB 2.0 Conference), varios expertos y grandes empresas decidieron que para sacarle mayor partido a la red de redes debían cambiar su punto de vista: nació la Web 2.0.
n 
nComo Ross Mayfield (directivo de SocialText) acuñó entonces: Web 1.0 era para el comercio, Web 2.0 es para la gente.
n 
nA raíz de estas reuniones surgieron nuevos conceptos y tendencias que pronto se verían reflejados en Internet.
nLa Web 2.0 surge como una actitud y no como una nueva tecnología. Un nuevo modo en el que el usuario es el que controla la información. Según esta nueva filosofía, para el empresario un sitio web deja de ser suyo para pasar a ser nuestro. Representa la evolución de las aplicaciones tradicionales hacia las aplicaciones web orientadas al usuario. Unas aplicaciones que generen y sean generadas por la colaboración, que presten servicios reales para poder reemplazar (a largo plazo) las aplicaciones de escritorio (Office, Photoshop, Dreamweaver, etc...).
n 
nLa Web 2.0 no se trata de una nueva versión, sino de una nueva forma de hacer las cosas.
